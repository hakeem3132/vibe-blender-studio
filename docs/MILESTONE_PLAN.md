# Milestone Plan

1. **Foundation Beta — complete:** secure connection, stable IDs, inspection,
   bounded object tools and exact application transactions.
2. **Core animation — real workflow implemented:** bounded materials, lights,
   camera, object/camera/scalar keyframes, preview, image sequences and verified
   MP4. The overall status remains Foundation Beta until a current clean-install
   run can reach the package index; interactive visual validation and broader
   preset families remain follow-up beta work.
3. **Prompt intelligence:** provider-assisted structured planning, target
   resolution and preserve correction while retaining the deterministic offline
   baseline.
4. **Video:** storyboard, VSE, camera cuts, narration, subtitles and music.
5. **Character:** licensed fixture, rig validation, retargeting, deformation,
   motion cleanup and GLB export.
6. **Game:** validated Blender-to-Godot runnable prototype.
7. **Launch audit:** fresh install, security, licensing, performance and release
   acceptance.

Milestone 2 does not begin character, Godot, ComfyUI, FreeMoCap, text-to-3D,
render-farm or collaboration work.
