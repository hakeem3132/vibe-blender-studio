# Repository Provenance

Vibe Blender Studio is derived from `PatrykIti/blender-ai-mcp` v3.3.0 at
upstream commit `43253155440f78ce208f7c4264bb8be6fb784ec7`. The upstream
Apache-2.0 attribution, license and notices remain part of this repository and
the packaged add-on.

During the product implementation, workspace maintenance removed the original
local Git object database. Consequently, the earlier product commit objects
`35e878a2d1d00fa66c35bf5d4cd58d4005951288`,
`45d856b297eb8c84e2fcd89fcc850e890d434487` and
`cfec314b17c9b7fbbe22cf70b5f416e5dfe080ac` became unavailable and cannot be
proved reachable from the preserved repository. They were not recreated with
fabricated identities.

The retained Foundation and animation source was truthfully re-anchored to the
pinned upstream commit as product snapshot
`15fbffb38b75a0d03c07e18e8f4548cbeaa56e4e`. Milestone 2.1 concludes at source
revision `f24b71f9b3fbac8625ba1f9620cf390557cc2874`; this provenance document is a
publication-only commit after that tested source revision.

The verified transfer bundle is `outputs/vibe-blender-milestone2_1.bundle` with
SHA-256
`670cc10672f5e9a77f9ff2eaf67a5115e1b8e81e71855f71c0ccb2e595ee7d91`.
At verification time it contained the complete reconstructed branch history and
pointed `codex/vibe-blender-launch` at the tested Milestone 2.1 revision above.

This record distinguishes preserved source evidence from unavailable historical
objects. It makes no claim that the missing commits were recovered or that their
original hashes were reconstructed.
